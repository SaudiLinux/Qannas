# دليل البدء السريع - أداة Qanas (قناص)

## مقدمة

هذا الدليل السريع يساعدك على البدء باستخدام أداة Qanas (قناص) في أقل وقت ممكن. للحصول على معلومات أكثر تفصيلاً، يرجى الرجوع إلى ملف README.txt ودليل الاستخدام الكامل.

## التثبيت السريع

### على نظام Windows

1. قم بتشغيل ملف `install.bat` بالنقر المزدوج عليه أو من خلال موجه الأوامر:
   ```
   install.bat
   ```

2. سيقوم السكربت تلقائيًا بإنشاء المجلدات اللازمة وتثبيت متطلبات Python.

3. سيتحقق السكربت أيضًا من وجود الأدوات الخارجية المطلوبة (nmap, whois, host) وسيخبرك إذا كانت غير موجودة.

### على نظام Linux

1. افتح الطرفية وانتقل إلى مجلد المشروع.

2. قم بتنفيذ سكربت التثبيت:
   ```
   chmod +x install.sh
   ./install.sh
   ```

## الاستخدام السريع

### فحص هدف واحد (موقع ويب أو عنوان IP)

```
python qanas.py example.com
```

هذا سيقوم بإجراء فحص شامل للهدف المحدد وحفظ النتائج في مجلد `loot`.

### فحص عدة أهداف من ملف

1. قم بإنشاء ملف نصي يحتوي على قائمة الأهداف (هدف واحد في كل سطر).

2. لإجراء فحص سريع (وضع AIRSTRIKE):
   ```
   python qanas.py path/to/targets.txt airstrike
   ```

3. لإجراء فحص شامل (وضع NUKE):
   ```
   python qanas.py path/to/targets.txt nuke
   ```

### عرض نتائج الفحص

```
python qanas.py loot
```

هذا سيقوم بإنشاء تقارير HTML للنتائج وعرضها في المتصفح الافتراضي.

## الأوامر الأساسية

| الأمر | الوصف |
|-------|-------|
| `python qanas.py example.com` | فحص هدف واحد (وضع عادي) |
| `python qanas.py path/to/targets.txt airstrike` | فحص سريع لعدة أهداف |
| `python qanas.py path/to/targets.txt nuke` | فحص شامل لعدة أهداف |
| `python qanas.py loot` | عرض نتائج الفحص |
| `python qanas.py --version` | عرض إصدار الأداة |
| `python qanas.py --help` | عرض المساعدة |

## نصائح سريعة

1. **تثبيت الأدوات الخارجية**: للحصول على وظائف كاملة، تأكد من تثبيت الأدوات التالية:
   - nmap: لفحص المنافذ
   - whois: لاستعلامات WHOIS
   - host: لاستعلامات DNS

2. **صلاحيات المسؤول**: قم بتشغيل الأداة بصلاحيات المسؤول للحصول على نتائج أفضل، خاصة عند إجراء فحص Nmap.

3. **اختيار وضع الفحص المناسب**:
   - الوضع العادي: لفحص هدف واحد بشكل مفصل
   - وضع AIRSTRIKE: لفحص سريع لعدة أهداف
   - وضع NUKE: لفحص شامل لعدة أهداف (يستغرق وقتًا أطول)

4. **تنظيم النتائج**: استخدم وضع LOOT بانتظام لتنظيم وتحليل نتائج الفحص.

## استكشاف الأخطاء وإصلاحها

### مشكلة: الأداة لا تعمل على Windows

**الحل**: تأكد من تثبيت Python 3.8+ وإضافته إلى متغير PATH. قم بتشغيل الأداة من موجه الأوامر أو PowerShell بدلاً من النقر المزدوج على الملف.

### مشكلة: رسائل خطأ "Command not found"

**الحل**: الأداة ستستمر في العمل مع وظائف محدودة وستقوم بتسجيل رسائل الخطأ في مجلد النتائج. قم بتثبيت الأدوات المطلوبة (nmap, whois, host) لتمكين جميع الوظائف.

### مشكلة: عدم وجود مجلد loot

**الحل**: قم بإنشاء مجلد `loot` يدويًا في مجلد المشروع أو استخدم سكربت التثبيت.

## الخطوات التالية
