# سجل التغييرات - أداة Qanas (قناص)

## الإصدار 1.0.1 (2025-06-25)

### تحسينات
- إضافة وظيفة `check_command_exists()` للتحقق من وجود الأوامر الخارجية قبل تنفيذها
- تحسين وظيفة `run_command()` للتعامل مع حالات عدم وجود الأوامر الخارجية
- إضافة معالجة أخطاء لوظائف الفحص الرئيسية (Nmap, WHOIS, DNS)
- إضافة تسجيل الأخطاء في ملفات نصية داخل مجلد النتائج
- إضافة ملف `install.bat` لتسهيل عملية التثبيت على نظام Windows
- إضافة ملفات توثيق شاملة باللغة العربية

### إصلاحات
- إصلاح مشكلة عدم وجود مجلد `loot` عند تشغيل الأداة لأول مرة
- إصلاح مشكلة توقف الأداة عند عدم وجود الأوامر الخارجية (nmap, whois, host)
- تحسين رسائل الخطأ لتكون أكثر وضوحًا للمستخدم

## الإصدار 1.0.0 (2025-06-01)

### الميزات الرئيسية
- إطلاق النسخة الأولى من أداة Qanas
- دعم أوضاع الفحص المختلفة (normal, airstrike, nuke, loot)
- دعم فحص الأهداف المفردة والمتعددة
- دعم فحص المنافذ باستخدام Nmap
- دعم استعلامات WHOIS للنطاقات
- دعم استعلامات DNS
- دعم فحص تطبيقات الويب
- إنشاء تقارير HTML تفاعلية
- دعم قوائم الكلمات المخصصة
- دعم الإضافات (plugins)

### المتطلبات
- Python 3.8+
- الأدوات الخارجية: nmap, whois, host
